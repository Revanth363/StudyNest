auth.middleware.js	Verifies JWT tokens, authenticates users, and attaches user 				information to the request object (req).
error.middleware.js	Centralized error-handling middleware that catches application errors 			and returns consistent API responses.
security.middleware.js	Enhances application security using Helmet, rate limiting, and NoSQL 			injection sanitization.
upload.middleware.js	Handles file uploads using Multer with in-memory storage before 			processing or uploading to Cloudinary.
